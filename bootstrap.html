Bootstrap 3: Images

Images can be styled in Bootstrap 3 using the following classes:
img-rounded: Adds rounded corners to the image.
img-circleCrops the image into a circular shape.
img-thumbnail Create a thumbnail.

To create a responsive image that resizes to fit the screen.add class img-responsive to the image tag.

Bootstrap 3: Forms & inputs

A form is for receiving user input. It can include inputs, selectable list, radio buttons, and checkboxes. And vertical form is the default in bootstrap and can be created as follows:
1. Wrap the entire form in a form element.
2. Use a div of class form-group to wrap input, textarea, and select lists elements. Use a form-control class for all textual input elements.
3. To add a checkbox or radio button, create a label tag with coresponding input inside it. They can be placed in a div of class checkbox or radio respectively.
4. To display a checkbox or radio button group in a single line, do not use a div and add checkbox-inline or radio-inline class to all the label elements in the group.

Bootstrap supports HTML5 input types email (as used in code above), text, password, datetime, datetime-local, date, month, time, week, number, url, search, tel and color.
An inline form includes class form-inline in the form element an displays all form elements in a single line. An inline form transforms into a vertical form for screens less than 768px wide.

Bootstrap 3: Dropdowns

A dropdown menu allows the user to choose one value from a predefined list which drops down on clicking.
To create a basic Bootstrap dropdown:
1. Create a div element with dropdown class.
2. Create a button element with dropdown-toggle class an data-toggle="dropdown" attribute so that the dropdown menu opens on clicking it.
3. Create a span element with caret class within the button element to display a caret arrow icon.
4. Create a ul element with dropdown-menu class to define the dropdown options.

Addiotional styling can be added with these classes for the li elements of the ul element.

divider

Adds a divider.

dropdown-header

Adds a header.

active

Adds an enabled dropdown item.

disabled

Adds a disabled dropdown item.

Bootstrap 3: Progress Bars

A basic a Bootstrap progress bar can be created as follows:
1. Create a div element with progress class.
2. Inside in progress div just created, create a div element with a progress-bar class and role="progressbar" property. Sets its CSS width property to the progress values of the progress bar. For example, style="width:59%".

The value, min, max attributes of the general HTML progress bar should be aria-valuenow, aria-valuemin, aria-valuemax attributes, respectively.

Additional styling can be added with these classes: progress-bar-info, progress-bar-success, progress-bar-warning, progress-bar-danger, progress-bar-striped.

To create a stacked progress bar, place multiple progress bars in the same div class="progress" element.

The aria-* attributes are not required, but when aviable, the Accesible Rich Internet Applications technology can be interpreted be assistive technologies, such as screen readers.
